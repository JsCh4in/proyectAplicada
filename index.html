Sistema de seguridad

El sistema de seguridad funciona a traves de un sensor biometrico de escaneo de huella dactilar que nos permite reconocer al personal autorizado, una vez reconocido el usuario se abrira la puerta o compuerta, a su ves se notificara al dueño o personal de seguridad quien ha ingresado al area de trabajo, tambien cuenta con una alerta de sonido para saber que la puerta esta abierta, dado el caso de no reconocer la huella, se enviara una alerta al personal de seguridad y tendra una alerta por sonido.

Información del producto

Hardware

Sensor de huellas controlado a traves de un microcontrolador Esp-32, con capacidad de almacenar mas de cien huellas de diferentes usuarios, tendra alertas de sonido mediante un buzzer y alertas de seguridad via telegram.

Software

Programado en micropython o c++, nos permite comunicar la informacion del microcontrolador con una pequeña base de datos que almacena y compara las huellas digitales de los usuarios.

Implementación

Apto para sitios donde el personal es restringido o solo ciertas son permitidas, tanto a nivel de empresa como a nivel personal o de hogar.
